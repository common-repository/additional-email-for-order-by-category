Additional Email for order by Categories 

== Description ==
Send additional Email for each order by Wocommerce category. Send email when order contain a product that from selected category. It will trigger when order is placed. (woocommerce_checkout_order_processed action hook). Message body will be your custom message with order details 

== Installation ==
Just upload and install this plugin
You can access settings from Settings -> Additional Email Settings 
Url is domain.com/wp-admin/options-general.php?page=additional-email-setting-admin
